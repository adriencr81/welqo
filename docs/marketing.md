## Landing Page — Structure exacte

### Hero
**Headline** : "Le livret d'accueil digital que vos voyageurs adorent ouvrir"
**Sous-titre** : "Créez un guide élégant en 15 minutes. Vos touristes le consultent hors-ligne, sans app, sans compte. Plus jamais le même message à 23h."
**CTA principal** : "Obtenir l'accès à vie — 59€" (early bird)
**Social proof sous le CTA** : "Rejoignez 47 hôtes qui ne répondent plus aux mêmes questions" (mettre à jour dynamiquement)

### Problème (section "Vous connaissez ça ?")
- 📱 "C'est quoi le code Wi-Fi ?" — à 23h, le soir de l'arrivée
- 🧺 "Comment fonctionne la machine à laver ?" — le lendemain matin
- 🗑️ "Où sont les poubelles ?" — le jour du départ
- 🍽️ "Un restaurant ouvert le dimanche ?" — 3 fois par mois

**Texte** : "Vous avez déjà répondu à ces questions des dizaines de fois. Un guide numérique règle ça une fois pour toutes."

### Démo produit (GIF ou vidéo 30s)
Montrer : création d'une section Wi-Fi → scan QR → affichage sur téléphone

### Features (3 colonnes)
1. **Ultra-simple à créer** — "15 minutes et votre guide est prêt. Pas de formation, pas de tutoriel."
2. **Fonctionne hors-ligne** — "Vos touristes sont dans un village sans réseau ? Le guide s'ouvre quand même."
3. **À vie, pas par mois** — "Payez une fois, utilisez pour toujours. Pas d'abonnement surprise."

### Comparaison (tableau)
| | TouchStay | Hostfully | Welqo |
|--|-----------|-----------|-------|
| Prix | 99€/an | 300€/an | **79€ à vie** |
| Mode hors-ligne | ✗ | ✗ | **✓** |
| Setup | 1-2h | Demi-journée | **15 min** |
| Multilingue | Payant | Payant | Post-MVP |

### Témoignages (à obtenir des bêta-testeurs semaine 1)
Format : photo + prénom + ville + nombre de propriétés + citation

### FAQ
**Q : Est-ce que mes voyageurs doivent télécharger une app ?**
R : Non. Ils scannent le QR code et le guide s'ouvre directement dans le navigateur.

**Q : Que se passe-t-il si j'ai plusieurs logements ?**
R : Vous créez autant de guides que vous voulez, chacun avec son propre QR code.

**Q : Est-ce que le guide fonctionne sans connexion internet ?**
R : Oui. Dès que le guide a été chargé une fois, il reste accessible hors-ligne.

**Q : Que se passe-t-il si je modifie mon guide après que le touriste l'ait chargé ?**
R : Ils voient les modifications dès qu'ils se reconnectent ou rechargent la page.

**Q : C'est vraiment à vie ? Pas de frais cachés ?**
R : À vie signifie à vie. Pas d'abonnement, pas de frais par propriété, pas de surprise.

### CTA final
"Rejoignez les hôtes qui ont arrêté de répondre aux mêmes questions — 59€ une fois"
_(early bird — prix monte à 79€ après les 50 premières places)_
